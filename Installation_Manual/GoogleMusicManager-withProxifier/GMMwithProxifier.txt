OK 借助Proxifier

-


联网
```
小小办法（失败）
Proxifier 退出
ssr PAC模式 / 全局模式
(没有代理设置)(代理 1080)

终极办法
Proxifier Default 覆盖
Proxifier +MusicManager.app 1080
ssr PAC模式
(没有代理设置)代理 自动检测

```

说明1：
ssr PAC模式和全局模式 不能支持 GMM 登入
说明2：
Proxifier 用最大性能 开Default覆盖，Rules: "Default覆盖,MusicManager.app" => 支持 GMM 登入，支持 GMM 上传
说明3：
如果出问题，1.校准电脑时间 2.换ss帐号
说明4：
GMM 开机运行时，(不)必须启用 Proxifier-Default覆盖 (只要登入了就能上传歌曲)

troubleshooting:
If any of your lyrics errored out from Lyrics Panel trying to download, you should see text files in your folder with the following text: <?xml version="1.0" encoding="UTF-8" ?> <result errmsg="Search ID or Code error!" errcode="32006"></result> Delete these erroneous text files and you should display the proper lyrics if you have a matching .lrc file for your song.

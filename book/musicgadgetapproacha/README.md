
说明 (MIUI x 红米手机3)：
- 重置手机与 ROM 版本
- - 重置手机 并不会改变当前的 ROM 版本 (不会恢复到 本手机刚买的时候的自带的版本)

- 重置手机与 硬盘 (手机内部储存, SD卡储存)
- - 重置手机 会 清除 手机内部储存 , 并重置
- - 重置手机 不会 清除 SD 卡的文件

- 刷机 卡刷 与 BL锁
- - 刷机目的：给 机子 得到新 ROM (新版本的 ROM)
- - ROM 分类(稳定版 / 开发版, 国际版)
- - 机子分类 (天生不带BL锁 / 天生带BL锁, 带BL锁但已解锁)
- - 刷机办法分类(线刷, 卡刷)
- - 得到新ROM(OTA自动升级 / 人工刷机)的结果 (原root失去)
